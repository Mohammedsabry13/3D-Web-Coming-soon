credit is required: github.com/Mohammedsabry13

A visually striking **"Coming Soon"** / landing page template for creative digital agencies, featuring an immersive interactive 3D space sphere background powered by Three.js.

## 🛠️ Tech Stack
- **HTML5 & CSS3** — semantic structure and custom styling
- **Bootstrap 5.3** — responsive layout
- **Three.js (r128)** — interactive 3D animated sphere background
- **Vanilla JavaScript** — section slider, options panel, form validation

## ✨ Features
- Interactive Three.js 3D space sphere as a full-page background
- Fullscreen vertical section slider (Hero → About → Contact)
- Bullet dot navigation between sections
- Rotating hero headlines with smooth transitions
- Live configuration panel (overlay scheme, animation type, bullet nav toggle)
- Email subscription form with validation
- Contact form with custom styling
- Social media icon links
- Fully responsive across all screen sizes
